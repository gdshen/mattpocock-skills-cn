# 分诊标签（Triage Labels）

这些 skills 使用五个 canonical triage roles 表达状态。此文件把这些 roles 映射到此 repo issue tracker 中实际使用的 label strings。

| mattpocock/skills 中的 label | 我们 tracker 中的 label | 含义 |
| ---------------------------- | ----------------------- | ---- |
| `needs-triage`               | `needs-triage`          | Maintainer 需要评估此 issue |
| `needs-info`                 | `needs-info`            | 等待 reporter 提供更多信息 |
| `ready-for-agent`            | `ready-for-agent`       | 已完整指定，可交给 AFK agent |
| `ready-for-human`            | `ready-for-human`       | 需要人类实现 |
| `wontfix`                    | `wontfix`               | 不会处理 |

当某个 skill 提到 role（例如“apply the AFK-ready triage label”）时，使用此表中对应的 label string。

编辑右侧列，使其匹配你实际使用的词汇。
